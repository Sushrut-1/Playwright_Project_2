# Playwright Automation Framework

## Project Overview

This project is an end-to-end test automation framework built using Playwright, JavaScript, and Cucumber (BDD). It follows the Page Object Model (POM) design pattern for better maintainability and scalability.

## Technologies Used

- Playwright
- JavaScript
- Cucumber BDD
- Node.js
- Git & GitHub

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Install Playwright browsers:

```bash
npx playwright install
```

## Running Tests

Run all test cases:

```bash
npx cucumber-js
```

## Framework Features

- BDD using Cucumber
- Page Object Model (POM)
- Cross-browser testing support
- Screenshot capture on failure
- Reusable page methods
- Easy maintenance and scalability
